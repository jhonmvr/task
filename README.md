# prueba

## Build from source

1. Clone the repo



2. Install dependencies.

   ```sh
   npm install
   ```

3. Build the production server.

   ```sh
   npm run build
   ```

4. Run the server.
   ```sh
   npm start
   ```

## Build Docker image locally

```sh
docker build -t prueba .
```

## Run tests

```sh
npm test
```
